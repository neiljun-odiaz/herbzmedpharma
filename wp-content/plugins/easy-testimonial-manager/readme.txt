== Description ==

Easy Testimonial Manager Plugin allows you to easily display testimonials along with Photo, Name, Company, URL and social profiles.  
Embed in any post/page using short-code <code> [easy-testimonial-manager]</code> or you can even drag and drop widget name called  
Easy Testimonial Manger at Widget setting.

= Features =
* Easy to use
* Fully responsive
* Auto fading testimonial
* Navigation bullets and buttons
* Simple and light weight
* 16 Different Social Icons
* Clean and Nice user interface
* Can be add title, social links and testimonial text, company along with url and social profiles.
* You can adjust the testimonial using settings.

= Demo =

[Click here](http://plugins.jwthemes.com/easy-testimonial-manager/) to see Easy Testimonial Manager in live action.

== Installation ==

1. Upload the `Easy Testimonial Manager` folder to the `/wp-content/plugins/` directory
1. Activate Easy Testimonial Manager Plugin through the 'Plugins' menu in WordPress
1. Configure the plugin by going to the `Easy Testimonial Manager` tab that appears in your admin menu
1. Add to any page using short-code ` [easy-testimonial-manager] ` or you can use widget too.
